# ✈️ Airport Delay Analytics Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard analyzing airline delay patterns across major airports operated by the Port Authority of New York and New Jersey.

The dashboard provides insights into flight delays, airline performance, and seasonal delay trends.

---

## Objectives
- Analyze flight delay patterns
- Compare airline performance
- Identify peak delay months
- Monitor cancellation statistics
- Provide interactive filtering for decision making

---

## Key Metrics
- Total Flights
- Total Cancelled Flights
- Average Arrival Delay
- Average Departure Delay
- Delay Severity Distribution

---

## Dashboard Features
- Monthly delay trend analysis
- Airline delay comparison
- Origin and destination filtering
- Interactive data exploration
- Geographic flight mapping

---

## Tools Used
- Power BI
- Data Modeling
- DAX Measures
- Data Visualization
